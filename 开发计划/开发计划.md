## 技术实现建议

### 数据库表结构（新增）

```python
# 文件表
class File(Base):
    id: int
    parent_id: int  # 自关联，支持层级
    name: str
    type: str  # 'file' | 'folder'
    content: str
    owner_id: int
    created_at: datetime
    updated_at: datetime

# 工作流表
class Flow(Base):
    id: int
    name: str
    description: str
    nodes: JSON  # 节点配置
    edges: JSON  # 连接关系
    owner_id: int
    created_at: datetime
    updated_at: datetime

# 执行记录表
class Execution(Base):
    id: int
    flow_id: int
    status: str  # pending/running/success/error
    result: JSON
    logs: JSON
    created_at: datetime
```

### 推荐使用的库

| 功能 | 库 |
|-----|---|
| 文件树 | `react-arborist` 或自研递归组件 |
| Flow 编辑器 | `reactflow` |
| Markdown 编辑 | `@monaco-editor/react` 或 `react-markdown` |
| 状态管理 | Zustand（已使用） |
| WebSocket | `socket.io` |
