# 虚拟文件系统开发计划

## 一、功能概述

### 1.1 核心目标
构建一个虚拟文件系统，支持用户在浏览器中创建、编辑、管理文件和文件夹，文件内容存储在数据库中，支持与工作流节点进行交互。

### 1.2 设计方案
采用**侧边栏 + 拖拽创建节点**的方案：
- 左侧：文件系统树形结构（独立组件）
- 中间：React Flow 画布
- 右侧：文件/节点属性编辑器
- 拖拽文件到画布自动创建文件节点

## 三、数据库设计

### 3.1 files 表

```sql
CREATE TABLE files (
    id SERIAL PRIMARY KEY,
    parent_id INTEGER REFERENCES files(id) ON DELETE CASCADE,
    name VARCHAR(255) NOT NULL,
    type VARCHAR(20) NOT NULL,  -- 'file' | 'folder'
    content TEXT,                -- 文件内容（文件夹为 NULL）
    owner_id INTEGER NOT NULL REFERENCES users(id),
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP,
    
    -- 索引
    INDEX idx_files_parent_id (parent_id),
    INDEX idx_files_owner_id (owner_id),
    INDEX idx_files_type (type)
);
```

### 3.2 SQLAlchemy 模型

```python
# app/modules/files/models.py
from sqlalchemy import Column, Integer, String, Text, ForeignKey, DateTime
from sqlalchemy.orm import relationship
from sqlalchemy.sql import func
from app.core.database import Base

class File(Base):
    __tablename__ = "files"

    id = Column(Integer, primary_key=True, index=True)
    parent_id = Column(Integer, ForeignKey("files.id", ondelete="CASCADE"), nullable=True)
    name = Column(String(255), nullable=False)
    type = Column(String(20), nullable=False)  # 'file' | 'folder'
    content = Column(Text, nullable=True)
    owner_id = Column(Integer, ForeignKey("users.id"), nullable=False)
    created_at = Column(DateTime(timezone=True), server_default=func.now())
    updated_at = Column(DateTime(timezone=True), onupdate=func.now())

    # 关系
    parent = relationship("File", remote_side=[id], backref="children")
    owner = relationship("User", backref="files")
```

## 七、拖拽功能设计

### 7.1 拖拽流程

```
1. 用户在文件树中按下文件项
   └─▶ 设置 drag data (文件信息)

2. 拖拽到画布区域
   └─▶ 画布显示放置提示

3. 用户释放鼠标
   └─▶ 触发 onDrop 事件
       └─▶ 弹出选择对话框 (Reader / Writer)
           └─▶ 创建对应类型的文件节点
```

### 7.2 数据传递

```typescript
// 拖拽开始时设置数据
const onDragStart = (event: DragEvent, file: File) => {
  event.dataTransfer.setData('application/json', JSON.stringify({
    type: 'file',
    id: file.id,
    name: file.name,
    path: file.path
  }));
};

// 放置时读取数据
const onDrop = (event: DragEvent) => {
  const data = JSON.parse(event.dataTransfer.getData('application/json'));
  if (data.type === 'file') {
    // 创建文件节点
    createFileNode(data);
  }
};
```
