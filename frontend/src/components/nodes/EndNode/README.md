# 结束节点组件

包含结束节点的所有相关组件：
- 画布中的节点显示组件
- 节点配置组件
- 节点编辑弹窗组件

## 组件列表

### EndNode.tsx
画布中结束节点的显示组件，包含节点的视觉样式和连接点。

### EndNodeConfig.tsx
结束节点的右侧配置面板，提供节点的详细配置界面。

### EndNodeEditor.tsx
结束节点的编辑弹窗，通过右键菜单打开，提供完整的节点编辑功能。

## 使用方式

```tsx
import EndNode from "./EndNode";
import EndNodeConfig from "./EndNodeConfig";
import EndNodeEditor from "./EndNodeEditor";
```
