# 选择节点

选择节点是一个智能节点，用于根据 AI 分析结果选择不同的输出路径。

## 功能特点

- **上侧输入**：可接收多个输入节点的数据
- **下侧输出**：可连接多个输出节点
- **AI 选择**：通过提示词让 AI 分析输入数据并选择合适的输出节点
- **灵活配置**：支持自定义提示词，可插入输入变量

## 工作原理

1. 收集所有上侧输入的数据
2. 将数据插入到提示词中
3. 调用 AI 分析并选择一个输出节点
4. 将输入数据传递给选择的输出节点

## 配置项

- **节点名称**：节点的显示名称
- **选择提示词**：指导 AI 如何根据输入数据选择输出节点的提示词

## 使用示例

### 基本用法
1. 创建选择节点
2. 在上侧连接多个输入节点（如用户输入、文件读取等）
3. 在下侧连接多个输出节点（如不同的处理逻辑）
4. 配置提示词，指导 AI 如何根据输入选择输出

### 提示词示例

```
根据用户的输入内容，选择最合适的处理路径：

用户输入：{input_123}

如果输入包含"紧急"或"重要"，选择节点 A
如果输入包含"查询"或"信息"，选择节点 B
否则选择节点 C

请仅返回节点 ID：A、B 或 C
```

## 注意事项

- 提示词应明确指导 AI 如何选择输出节点
- 确保下侧连接的输出节点有明确的标识
- AI 会返回选择的节点标识，系统会自动将数据传递给对应节点
